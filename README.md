# ArenaTicket Frontend

Frontend independente para venda de bilhetes ArenaTicket.

- Identidade visual própria (azul, branco/bege, sem referências a clubes)
- Experiência de compra rápida e minimalista
- Integração com backend via endpoints públicos (ver /FRONTEND_INTEGRATION.md)

## Estrutura inicial

- `src/` — código-fonte
- `public/` — arquivos públicos
- `docs/etapa-1/` — documentação da Etapa 1

## Dependências externas
- Nenhuma dependência visual ou fonte externa instalada nesta etapa.
- Qualquer dependência futura será documentada e aprovada previamente.

## Como rodar (após implementação)

1. Instale dependências: `npm install`
2. Rode o projeto: `npm run dev`

## Observações
- Nenhum arquivo legado ou referência a clubes.
- Todos os ativos visuais são neutros.
