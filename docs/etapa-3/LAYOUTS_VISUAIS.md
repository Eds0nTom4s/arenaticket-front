# Layouts Visuais — ArenaTicket UI

## Layout Base (BaseLayout.vue)

### Header
- Logotipo ArenaTicket (texto estilizado ou ícone azul)
- Botão de menu responsivo (hamburger)
- Menu lateral abre com links simulados (ex: "Meus Bilhetes", "Ajuda")

### Footer
- Texto central: “© ArenaTicket 2025 – Todos os direitos reservados”
- Versão da aplicação visível (ex: v1.0.0)

### Área Principal
- Conteúdo centralizado em grid
- Fundo branco ou bege claro (`--color-secondary`)
- Containers com sombra suave (`--shadow-card`)
- Espaçamentos uniformes conforme tokens

### Responsividade
- Mobile: 1 coluna
- Tablet: 2 colunas
- Desktop: 3 colunas
- Testado em 320px, 768px, 1366px

### Hierarquia Visual
- Header fixo no topo
- Footer fixo ou ao final da página
- Área principal ocupa o centro, com grid adaptativo

### Observações
- Layout minimalista, sem excesso de informação
- Navegação e foco acessíveis por teclado
- Nenhum dado real nesta etapa, apenas placeholders
