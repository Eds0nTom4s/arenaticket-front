# Especificação — Feedback Visual (ArenaTicket UI)

## Toasts
- Tipos: sucesso, erro
- Localização: canto superior direito
- Animação: entrada com slide + fade
- Duração: 3–5 segundos
- Mensagens simuladas:
  - “Compra simulada com sucesso”
  - “Ocorreu um erro”
- Ícone correspondente ao tipo
- Desaparecem automaticamente

## Loader
- Loader minimalista: círculo girando azul
- Centralizado na tela
- Não bloqueante
- Mensagem: “Processando pagamento...”

## Observações
- Todos os feedbacks são simulados nesta etapa
- Acessíveis por leitores de tela (aria-live)
- Contraste e tamanho testados para visibilidade
