# Variantes Visuais — ArenaTicket UI

## Combinações Recomendadas
- **AtCard + AtBadge + AtButton:**
  - Card exibe badge de status (ex: "Disponível") e botão de ação
- **AtModal + AtButton:**
  - Modal com botões de ação no rodapé
- **AtToast + AtIcon:**
  - Toast com ícone correspondente ao tipo de mensagem

## Padrões de Espaçamento
- Utilizar tokens de espaçamento (--space-*) entre componentes
- Margens e paddings consistentes conforme o layout

## Hierarquia de Cores e Contraste
- Títulos sempre em Poppins
- Corpo em Inter
- Contraste mínimo 4.5:1 para textos principais
- Botões e CTAs com contraste máximo

## Regras de Consistência
- Estados de hover ≠ focus (diferenciados visualmente)
- Títulos sempre em Poppins, corpo em Inter
- Não misturar variantes de botão no mesmo contexto visual

## Observações
- Todas as combinações devem ser validadas visualmente antes de uso em produção.
