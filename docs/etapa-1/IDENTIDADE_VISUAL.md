# Identidade Visual — ArenaTicket

## Nome da aplicação
ArenaTicket

## Paleta de Cores
- Azul primário: `#0066FF`
- Azul escuro: `#001B33`
- Bege claro (secundário): `#F9F7F4`
- Branco: `#FFFFFF`
- Cinza suave: `#DFE3E6`
- Verde-água (acento, uso restrito): `#2DD4BF`

## Tipografia
- **Títulos:** Poppins Bold
- **Corpo:** Inter Regular
- **Fallback:** sans-serif

## Tokens Essenciais
- Espaçamentos: xs, sm, md, lg
- Border radius: `--radius-card: 16px`
- Sombra card: `--shadow-card`
- Transição padrão: `--transition: 0.22s ease`

## Regras Visuais Fixas
- Cards limpos com borda neon azul suave (glow moderado)
- Animações curtas e só via transform/opacity (≤ 250ms)
- Contraste acessível (WCAG AA) para textos e CTAs

## Observações
- Não usar variações ad-hoc de cor.
- Todos os ativos visuais devem ser neutros, sem referência a clubes.
- Este documento deve ser validado pelo Product Owner antes de avançar para implementação.
