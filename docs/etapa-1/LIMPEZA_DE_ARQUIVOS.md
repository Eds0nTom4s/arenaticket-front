# Limpeza de Ficheiros — ArenaTicket

## Critério
Remover todos os arquivos e pastas que não estejam listados no README, arquivos de build, ou não sejam necessários para o novo projeto.

## Arquivos a remover (exemplo)
- Qualquer pasta ou arquivo legado de projetos anteriores
- Qualquer referência a clubes
- Qualquer arquivo não listado na estrutura mínima:
  - src/
  - public/
  - docs/etapa-1/
  - README.md
  - .gitignore
  - package.json
  - vite.config.js
  - FRONTEND_INTEGRATION.md (mantido para referência de integração)

## Observações
- Antes de remover, garantir que não há dependência funcional.
- Após limpeza, revisar com o PO.
